# 提问的智慧（原文）

::alert{type="info"}
本篇《提问的智慧》（*How To Ask Questions The Smart Way*）由 **Eric Steven Raymond** 与 Rick Moen 撰写，是开源社区流传最广的提问礼仪指南之一。为尊重原作者版权与协议，本站不搬运全文，仅提供权威出处索引。想读全文，请移步下方链接。
::

## 关于这篇文章

它系统地讲述了：在向技术社区求助之前应当做哪些自救（搜索、读文档、读 FAQ、自己实验），如何选择合适的提问场合、如何写出切中要害的标题与问题描述、如何避开经典的 "XY 问题"，以及求助后如何优雅地反哺社区。它的许多观点至今仍是网络求助的黄金准则。
