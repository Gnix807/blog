# 旧电脑突然暴毙，我在毕业季紧急换机

## 起因：那天它再也没亮起来

4 月 15 日，离毕业答辩还有两周，离论文终稿截止还剩 5 天。

上午正在撰写毕业论文，电脑嘈杂的风扇突然安静了下来，然后屏幕一黑再也没了动静。我按下电源键——风扇转了两秒停了，屏幕一片漆黑，连 Dell Logo 都没出现。再按。再长按。拔电源、拔外设、等十分钟再插电——死活不亮。充电指示灯倒是正常，但整台机器就像进入了某种不可逆的植物人状态。

::alert{type="warning"}
备份的重要性，永远是在出事之后才被深刻理解。好在旧机器的硬盘是完好的，否则这篇文章可能就是另一种基调了。
::

## 这台戴尔，跟了我四年

2022 年高考完购买了这台戴尔游匣 G15 5520，配置放在当时算中高端：

::card-list
- **CPU**：Intel Core i7-12700H（14核20线程）
- **GPU**：NVIDIA GeForce RTX 3070 Ti Laptop（8GB）
- **内存**：16GB DDR5 4800MHz
- **硬盘**：512GB NVMe SSD
- **屏幕**：15.6 寸 FHD IPS 165Hz
::

四年来它陪我跑过无数个建模渲染、视频剪辑、游戏开黑。老实说，3070 Ti 在 1080p 下至今依然战得动绝大多数游戏，剪辑 4K 素材虽然风扇起飞但也不至于卡死。戴尔的散热在中低负载下还算安静，键盘手感也说得过去——唯一让我不满的是那块 56Wh 的小电池，离了电源活不过两节课。

这台机器见证了不少重要时刻：

- **视频剪辑**：PR 剪 1080p 多轨非常流畅，4K 素材需要先代理，否则回放会偶尔卡顿。后来学会了用代理工作流，体验好了不少。DaVinci Resolve 跑得比 PR 更稳——可能因为它对 NVIDIA 的编解码器支持更好。
- **游戏**：《赛博朋克 2077》1080p 光追中特效大概 50-60 帧，关了光追能稳定 70+。《APEX》全高画质稳 144 帧以上。《博德之门 3》第三章人多的地方掉帧明显，降到 40-50，不过那游戏第三章谁来了都卡。
- **AI 相关**：3070 Ti 的 8GB 显存跑 Stable Diffusion 勉强够用。出 512×512 的图很快，一旦上高分辨率或者开 ControlNet 就开始爆显存。也是因为这个，去年试了几次本地跑 LLM 后就放弃了——7B 模型勉强跑，13B 直接 OOM。后来也试过本地的TTS，效果也差不多。

总体上，我对它的评价是：**皮实但不够精致，够用但不够惊艳**。

## 电脑城判死刑

抱着「也许只是电源模块烧了」的侥幸，当天下午就奔去了电脑城。

师傅拆开后盖，测了几个关键点位，沉默了一会儿，抬头说：「供电正常，但上电时序异常——大概率是主板芯片组的问题，更具体得查电路图。这机器过保了，修的话不保证能好，换主板至少两千多，而且你这种老机型不一定有备件。」

一句话：**不如不修**。

当时我跟他说我论文还有一周截止。师傅的表情很微妙——类似「那你来错地方了，应该去隔壁京东」。

从电脑城出来，给家里打了个电话。我本来做好了据理力争「为什么修不如换」的准备，没想到我妈听完就说了一句：「论文要紧，赶紧买新的，钱不够我们给你补。」挂了电话在路边呆站了几秒——这台旧电脑当初也是他们掏钱给我买的，几年后又是因为学业需要，二话不说就答应再买一台。说实话，那一刻比电脑坏了更让我难受。

## 没时间纠结了

当场蹲在电脑城门口的长椅上打开京东，开始筛选。需求很明确：

::card-list
- **必须现货**：等不起预售
- **性能不能降**：旧机是 i7 + 3070 Ti，换机不能开倒车
- **预算 1 万左右**：毕业季花销大，但也没法省
- **屏幕要好**：论文排版和后期调色都对屏幕有要求
::

戴尔新游匣 G16 其实看了，但全都是40系的显卡，而且游匣整条产线都已被戴尔取消并入了外星人，热评里也是一堆吐槽散热和做工缩水的，果断跳过。拯救者 Y9000P 是好机器，但 i9 + 5070 的配置超预算不少。暗影精灵价格合适，但那句「鼓包精灵」的外号实在让我下不了决心。

::folding
#title
各备选机型的取舍
#default

- **戴尔游匣 G16**：同品牌换机本来是最省事的——硬盘拆下来直接插上就能用，系统都不用重装。但翻了翻京东和 B 站的评价，G16 和 G15 真不愧是一对笑面虎，两头乌角鲨。还有一个细节：G16 用的也是倒装主板，自己清灰换硅脂极其麻烦。已经被戴尔的售后和清灰折腾够了，不想再来一次。
- **联想拯救者 Y9000P 2026**：各方面最水桶的选择，做工、屏幕、散热都没短板，售后网点也多。但 i9-14900HX + RTX 5070 的配置京东自营标价要一万五了，国补后也要一万二三，加上内存只有 16G 必须自己扩容，总花费轻松破绷。毕业季用钱的地方多，不想在电脑上超预算。
- **惠普暗影精灵 10**：价格非常香，但暗影精灵有一个祖传问题——电池鼓包。惠普官方对此一直装死，论坛上每隔几天就有人发鼓包照片。买一台随时可能物理膨胀的电脑有点太刺激了，pass。
- **ROG 魔霸新锐**：ROG 的做工和颜值确实高，但那颗败家之眼的溢价太狠了。同配置比天选贵了近两千，多的钱主要花在 RGB 灯效和 A 面 logo 上，对我这种不爱灯效的人来说纯属浪费。
- **机械革命旷世**：性价比拉满，同配置也是相对来说比较便宜的。但品控全靠抽奖，售后体系也不如一线品牌。平时折腾一下无所谓，毕业季这种关键时刻赌不起。
::

最后落在了**华硕天选 6 Pro**上。9955HX + 5070 + 16 寸 2.5K 屏，国补后 8999，华硕的品控至少比二线品牌靠谱，售后网点也算齐全。在紧急状态下，这是一个不需要犹豫的组合。

为什么会选它？几个理由：

1. **9955HX**——AMD 这一代移动旗舰，16 核 32 线程的规格放桌面端都算中高端，跑我的剪辑和渲染绰绰有余
2. **RTX 5070**——比旧机 3070 Ti 强了两代，且支持 DLSS 4，打游戏和 AI 推理都有提升
3. **16 寸 2.5K 165Hz**——终于不用在 1080p 下看 PPT 了，高刷日常滚动网页都很舒服
4. **现货 + 国补后 8999**——京东自营显示「明日达」，价格卡在万元以内，不废话直接下单

::alert{type="tip"}
下单时看了一眼评论区，最打动我的是一条差评：”屏幕漏光明显，换了一次才好。“——敢晒差评的平台至少说明没有控评。买电子产品我一般都先看差评，好评可以刷，但真实痛点藏在中差评里。
::

## 开箱与上手

第二天一早，京东小哥抱着那个青色的箱子敲开了门。

拆开包装，天选 6 Pro 白色款的真机比官网渲染图更好看。A 面是磨砂金属，机甲风 logo 低调地印在上面，没有 RGB 光污染。整体设计比前几代收敛了不少，带出去开会也不至于被当成电竞少年。

新机配置一览：

::card-list
- **CPU**：AMD Ryzen 9 9955HX（16核32线程，Zen 5架构）
- **GPU**：NVIDIA GeForce RTX 5070 Laptop（8GB GDDR7）
- **内存**：16GB DDR5 5600MHz
- **硬盘**：1TB NVMe PCIe 4.0 SSD
- **屏幕**：16 寸 2560×1600 IPS，165Hz，100% sRGB，500nit
- **电池**：90Wh
::

开盖即开机，Windows 11 初始化很快。

几个让我惊喜的点：

**屏幕**。从 15.6 寸 1080p 到 16 寸 2.5K，感受到的不是参数，是视力被尊重。论文里的小字终于不用凑近屏幕看了。这块屏标称 100% sRGB，官方规格里没有标 DCI-P3，实际色彩表现中规中矩——剪个视频、修个图完全够用，但如果有专业调色需求还是建议外接显示器校准。165Hz 刷新率日常用很丝滑，G-Sync 打游戏也能减少撕裂。

**触控板**。戴尔的触控板属于「能用，但不如直接用鼠标」。天选 6 Pro 的触控板大了一整圈，玻璃材质，滑动顺畅，点按反馈清晰。写论文那几天，大部分时间都不需要鼠标。

**续航**。90Wh 的大电池确实管用，图书馆写论文（Chrome + Word + Zotero，开省电模式）撑了将近 7 个小时——旧机同样场景不超过 2.5 小时。第一次体会到带笔记本出门不用满楼层找插座的自由。

### 外观与做工

天选 6 Pro 的三围是 35.4 × 26.9 × 1.79~2.73 cm，重约 2.3kg。作为一台 16 寸游戏本，这个体积控制得不错——得益于窄边框设计，整体尺寸比我的老款 15.6 寸戴尔还略紧凑一点，塞进书包不会觉得笨重。

A 面是金属磨砂材质，触感冰凉扎实。天选系列的机甲风 logo 印在上面，没有 RGB 灯效——远看就是一台普通的白色笔记本，完全没有电竞本惯有的「存在感溢出」。这在图书馆和教室场景下是个加分项。

C 面是塑料，但是手腕放上去比戴尔游匣那个大塑料舒服不少。不过白色机身有一个无法回避的问题——显脏。手托位置用久了会泛黄，两个月下来已经有轻微的变色痕迹。不算严重，但估计一年后会比较明显。

**键盘**。支持单区 RGB。1.7mm 键程，按压反馈干脆利落，回弹跟手。长时间写论文和敲代码，手感比 ThinkPad 的矮键帽更讨我喜欢。方向键是标准尺寸的（不是半高那种阉割版），数字小键盘也给全了——剪辑时设快捷键、打时间轴都很顺手。唯一想吐槽的是电源键：它在键盘右上角，和键盘好像是一个整体，前几代的机器如果电源键坏了要换一整个键盘，不知道华硕改没改。

**触控板**。面积比前代更大了，玻璃材质滑动顺畅，点按反馈清晰。写论文那几天，在不插鼠标的情况下居然没有不适感——这对 Windows 笔记本的触控板来说是相当高的评价了。

**接口**（以 9955HX + 5070 配置为准，从左到右）：

- 左侧：电源口 → RJ45 网口 → HDMI 2.1 → USB-C 3.2 Gen 2（支持 DP 输出 + PD 充电）→ USB-C 3.2 Gen 2（支持 DP 输出）→ USB-A 3.2 Gen 2 → 3.5mm 复合音频口
- 右侧：两个 USB-A 3.2 Gen 2

总共 3 个 USB-A + 2 个 USB-C + HDMI 2.1 + RJ45。作为剪辑用户，双 C 口意味着可以同时外接显示器和读卡器而不用扩展坞，RJ45 网口在天选全系保留也是加分项（不少轻薄游戏本已经砍掉了）。接口布局比戴尔游匣合理得多—— 戴尔经常把接口放在后方出风口旁边，而且接口规格也是阉割版的。

**摄像头与生物识别**。1080p FHD 摄像头，支持 Windows Hello 红外面部识别——开盖自动解锁，用起来很无感但习惯了就回不去。视频通话清晰度在游戏本里算中上，至少不会再被同学问「你摄像头是不是坏了」。

**其他细节**：
- 音响通过了 Hi-Res 认证，支持杜比全景声，外放音量够大但低音偏薄，看视频够用，听歌建议外接。
- Wi-Fi 6E + 蓝牙 5.4，连校园网和蓝牙耳机都很稳定，没掉过链子。
- 附赠的电源适配器是 280W 砖头——重量感人，出差党建议自备一个 100W PD 充电器应付轻度场景。

### 数据迁移与论文抢救

新机到手第一件事就先别跑分了，先恢复一下工作环境。

旧机的 512G SSD 拆下来，用 NVMe 硬盘盒连上新电脑。谢天谢地硬盘本身没坏——所有文件完整无缺。花了一下午做了这几件事：

1. **软件重装**：Adobe 全家桶、DaVinci Resolve、VS Code 等主力软件全部重装。用了 Chocolatey 包管理器，一条命令就把 Git、Node、Python、FFmpeg 等开发工具链拉回来了，省了大量逐个下载安装的时间。
2. **字体迁移**：做视频和排版的人最怕丢字体。旧系统里攒了两年多的几百款中英文字体全部从旧硬盘拷过来，Windows 的「拖进 C:\Windows\Fonts」虽然原始但有效。
3. **环境配置**：Git 全局配置、SSH 密钥、终端主题、VS Code 插件列表——这些从旧硬盘里翻出来，一一还原。好在做了备份清单，不至于漏掉什么。
4. **文章抢救**：说了这么多，最重要的论文一定要迁移过来，不然电脑不白买了吗。

::alert{type="tip"}
如果你有更换电脑的计划，**现在就**给自己的开发环境写一份 checklist。不需要多详细，列出你装了哪些软件、哪些插件、哪些字体就够了。真正换机的时候，这份清单能帮你节省至少半天时间。
::

### 实际使用感受

没有特意跑分，就从日常体验来说。9955HX 在平衡模式下几乎没有风扇声，跟我记忆中戴尔 i7-12700H 随便开个 Chrome 就起飞的状态差别巨大。PR 里回放 4K 多轨素材不再需要代理，DaVinci Resolve 导出速度明显快了一大截——同样的项目旧机要 4 分多钟，新机两分钟出头就搞定了。渲染这类吃多核的任务感受最直接：16 核 32 线程确实不是闹着玩的。

游戏方面，8GB 显存依然是 5070 的瓶颈，路径追踪全开的时候显存会吃紧，但日常游戏拉到高画质基本通吃。DLSS 4 的多帧生成在支持的几款 3A 里体验提升显著。AI 推理也是显存卡脖子——跑个 7B 的本地 LLM 够用，上 13B 就吃力了。真要正经搞 AI，可能还得是 128GB 统一内存的 MacBook，但那显然不是游戏本的战场。

## 新旧对比

从戴尔游匣到天选 6 Pro，说句公道话：

| 维度 | 戴尔游匣 G15 5520 | 天选 6 Pro |
|---|---|---|
| **CPU** | i7-12700H | R9 9955HX |
| **GPU** | RTX 3070 Ti | RTX 5070 |
| **屏幕** | 15.6 英寸 FHD 165Hz | 16 英寸 2.5K 165Hz |
| **内存** | 16GB DDR5 | 16GB DDR5 |
| **电池** | 56Wh（太拉了） | 90Wh |
| **重量** | ~2.5kg | ~2.3kg |
| **噪音** | 中高负载起飞 | 平衡模式很安静 |

::alert{type="warning"}
并不是说戴尔游匣一无是处——作为一款 2022 年的中高端游戏本，它在当时的综合表现对得起价格。只是四年后回头看，移动端的进步实在太快了：同样的尺寸重量，性能翻倍，续航翻倍，屏幕素质翻倍。不得不感叹摩尔定律在笔记本领域依然有效。
::

## 聊聊那台旧机器的结局

戴尔 G15 最终没有送去官方维修——师傅报价换主板 2800，而且不保证修好。四年机器残值可能也就两千出头，花比残值还多的钱去修，怎么看都不划算。

它现在安静地躺在防尘袋里，连同我的回忆一起。硬盘被我拆出来做了移动硬盘——数据都迁移到新机上了，但这块盘留着，权当一个纪念品。毕竟是人生第一台自己的游戏本，承载了大学四年里无数个通宵赶作业和开黑上分的夜晚。

## 写在最后

从旧机暴毙到新机到手，全程不到 36 小时。但这件事给我最大的教训不是「旧机器不可靠」，而是——

::quote
备份不是保险，是习惯。当你习惯了所有工作文件都活在 Git 和云盘里，电脑坏了就只是硬件问题，不会升级成生存危机。
::

如果你也临近答辩或项目交付，花十分钟检查一下：毕设代码 push 了吗？论文草稿在云端有一份吗？不需要等到电脑蓝屏那天再后悔。

至于天选 6 Pro——两个月用下来，综合做工、性能、屏幕和续航都算满意。唯一的小毛病是偶尔会在使用中自动重启，频率不高但确实存在，更新过几次 BIOS 和驱动后有所改善，不过没有彻底根除。在国补后 8999 这个价位，整体瑕不掩瑜。后续如果遇到值得写的使用体验，会再更新。
